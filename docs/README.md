# Pulsar

The Pulsar (in development) is an Adafruit feather compatible baseboard built with a multi-debug dbu onboard:
```
      .--------------------------------------------------------------------.
      | fizzit-pulsar                                                      |
      |          .------------------.                                      |
      |          | RK3308B Radxa S0 |                                      |
      |          |  Wifi & 10BaseT  |                                      |
      |    .-----|                  |------------.                         |
      |    |     `-----+------------'            |                         |
      |    |           |                         |                         |
      |--|USB|-------|TTY|---------------------|USB|-----------------------|
      |  `---'       `---'                     `---'                       |
      |    | Pwr       |                         |                         |
      |    |           |           .-------------------------.             |
      |    |           |           |          USB HUB        |             |
      |    |           |           `-------------------------'             |
      |    | .---.     |            |       |       |       |              |
     ---.  `-|USB|  .------. dap    |       |       |       |              |
     USB|----|SW |--| DBU  |------+-|-----+-|-----+-|-----+-|----.         |
     ---'    `---'  '------'      | |     | |     | |     | |     |        |
      |                | | uart   | |     | |     | |     | |     |        |
      |                | `------+-|-|---+-|-|---+-|-|---+-|-|---. |        |
      |                |        | | |   | | |   | | |   | | |   | |        |
      | 5V0            |        -----.  -----.  -----.  -----.  ---.       |
      |                |       /FZ1  | /FZ2  | /FZ3  | /FZ4  | /FZ5|     .---
      |                |       |     | |     | |     | |     | |   |     |--   8 x 4-way
      |                |       |     | |     | |     | |     | |   |     |--   FZ pin IO
      |                |       |      X       X       X       X    |     |--   board-to-wire
     ---.              |       |     | |     | |     | |     | |   |     |--   connectors
      - | 3V3          |       `-----' `-----' `-----' `-----' `---'     `---     
      - | 5V0          |          |       |       |       |      |         |
      - | VBAT         |          |6      |6      |6      |6     |3        |
      - | 0V           |          |       |    .- ----.   |      |         |
     ---'              |          |       `----|UP5K |----'      |         |
      |                |          `------------|FPGA |-----------'         |
      |                |                       `-----'  FZ pin IO          |
      |                |                   config |     X connect          |
      |                `--------------------------'                        |
      `--------------------------------------------------------------------'

```
